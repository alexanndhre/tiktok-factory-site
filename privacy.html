Política de Privacidade

Esta política descreve como o aplicativo "TikTok Factory Analytics" trata dados obtidos através da API do TikTok.

Quem usa este aplicativo

Este é um projeto pessoal de uso individual. Apenas a conta do próprio desenvolvedor está autorizada a usar este aplicativo — ele não é distribuído para outros usuários.

Quais dados são coletados

O aplicativo coleta, exclusivamente da própria conta autorizada: contagem de visualizações, curtidas, comentários e compartilhamentos dos vídeos publicados por essa conta.

Como os dados são usados

Os dados são armazenados em um banco de dados privado, controlado pelo próprio desenvolvedor, e usados unicamente para gerar relatórios internos de desempenho de conteúdo.

Compartilhamento de dados

Nenhum dado coletado é compartilhado, vendido ou repassado a terceiros sob nenhuma circunstância.

Retenção e exclusão

Os dados são mantidos enquanto o projeto estiver ativo. O desenvolvedor pode excluir todos os dados armazenados a qualquer momento.

Contato

Dúvidas sobre esta política podem ser enviadas ao desenvolvedor responsável pelo aplicativo.
